Projets & réalisations des élèves

La filière encourage projets techniques variés :

Exemples de projets possible en CIEL :

  • Caméra IP sécurisée — montage, configuration réseau, durcissement.
  • Serveur local — déploiement, sauvegardes, supervision (Zabbix / Prometheus exemples).
  • Robotique — châssis, capteurs, commandes, participation à défis.
  • IoT — capteurs environnementaux, dashboard en temps réel.
  • Projet chef-d'œuvre — projet annuel avec documentation, démonstration, rapport technique.
  • Pare-feu / VPN (pfSense / OPNsense) — installation, règles, VPN site-à-site et accès distant.
  • Audit de sécurité d’un poste — détection de failles, patching, rapport de recommandations.
  • Gestion de parc informatique — déploiement d’images, inventaire, scripts d’automatisation (SCCM / Ansible).
  • Serveur DNS / DHCP — configuration de zones internes, résilience, journalisation.
  • Application mobile simple — formulaire, stockage local, publication (Android / Flutter).
  • Mini-jeu — développement en Python ou JavaScript, scores, sauvegarde.
  • Système CRUD avec base de données — création, lecture, mise à jour, suppression (MySQL / SQLite).
  • Chat en réseau local — sockets, protocole simple, interface web ou console.
  • Intranet d’entreprise — gestion utilisateurs, partage de documents, authentification.
  • Site e-commerce simulé — catalogue, panier, administration, paiement fictif.
  • Transmission radio / IoT RF — modules 433 MHz / LoRa, envoi et réception de données.
  • Analyse de signaux audio — filtrage, spectrogramme, détection d’événements.
  • Automatisation domotique — MQTT, capteurs, scenarii (éclairage, volets).
  • Pilotage LED via web — interface de contrôle temps réel (WebSocket / HTTP).
  • Contrôle d’accès RFID — lecture badges, base d’utilisateurs, logs.
  • Station météo connectée — mesure T°C, humidité, pression, envoi sur serveur.
  • Compteur de passage — capteur infrarouge, statistiques, graphique web.
  • Suivi GPS — module GPS + transmission, tableau de suivi en ligne.
  • Montage vidéo promotionnel — tournage, prise de son, montage, export pro.
  • Studio d’enregistrement miniature — traitement audio, isolation, configuration micro.
  • Alarme maison DIY — capteurs d’ouverture, sirène, alimentation secours.
  • Joystick / contrôleur personnalisé — électronique, drivers, intégration PC.
  • Système de commande vocale — reconnaissance simple et exécution d’actions locales.
  • Tutoriels / documentation professionnelle — guides d’installation, procédures, schémas.
  • Simulation service IT — gestion des tickets, SLA, rapports d’intervention.
  • … et bien plus encore selon votre créativité et vos compétences en CIEL !
